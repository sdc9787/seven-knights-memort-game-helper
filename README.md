# Seven Knights Memory Helper (세븐나이츠 리버스 메모리 헬퍼)

세븐나이츠 리버스(Seven Knights Reverse) 내의 카드 뒤집기 미니게임을 위한 실시간 화면 분석 보조 도구입니다. Tesseract.js와 Canvas API를 활용하여 화면상의 카드를 인식하고, PiP(Picture-in-Picture) 모드를 통해 게임 화면 위에 가이드를 제공합니다.

## 🚀 주요 기능

- **실시간 화면 분석**: 브라우저의 화면 공유 기능을 통해 게임 화면을 캡처하고 실시간으로 분석합니다.
- **카드 인식 (OCR)**: Tesseract.js를 사용하여 카드에 적힌 텍스트를 인식하고 위치를 기억합니다.
- **PiP(Picture-in-Picture) 모드**: 분석된 카드 레이아웃을 별도의 팝업 창으로 띄워 게임 화면 위에 겹쳐서 볼 수 있습니다.
- **좌표 캘리브레이션**: 게임 해상도나 창 크기에 맞춰 인식 영역을 수동으로 세밀하게 조정할 수 있습니다.
- **로그 시스템**: 인식 상태 및 오류를 실시간으로 확인할 수 있는 로그 창을 제공합니다.

## 🛠 기술 스택

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animation**: Motion (framer-motion)
- **OCR**: Tesseract.js
- **Icons**: Lucide React

## 📖 사용 방법

1. **게임 실행**: 세븐나이츠 리버스 클라이언트를 **창모드**로 실행합니다.
2. **시작**: 웹 앱에서 '시작' 버튼을 누르고 화면 공유 팝업에서 **'창' -> '세븐나이츠 리버스'**를 선택합니다.
3. **캘리브레이션**: 게임 내 미니게임 화면이 나타나면, 설정 창에서 카드 영역이 정확히 일치하도록 좌표를 조정합니다.
4. **PiP 활성화**: 'PiP 모드 시작' 버튼을 눌러 가이드 창을 게임 화면 위에 배치합니다.
5. **플레이**: 카드를 뒤집으면 헬퍼가 자동으로 인식하여 위치를 표시해 줍니다.

## ⚠️ 주의 사항

- 본 사이트는 **크롬(Chrome)** 브라우저에 최적화되어 있습니다.
- 실시간 분석은 PC 자원을 소모하므로 성능에 따라 인식이 늦어질 수 있습니다.
- 마우스 포인터가 카드를 가리면 인식이 방해될 수 있으니 영역 밖으로 치워주세요.
- 기본 설정값은 최적화된 값이므로 가급적 유지하는 것을 권장합니다.
